Waitabit API

Enabling API Access

Open a API Information in Waitabit. Provide neccessary details in the given field Click on the Genetate API and copy the API key Read below for API endpoints.

Authentication

You need to be authenticated for all API requests. You can generate an API key in your developer dashboard. Add the API key in the Header.

Header

X-API-TOKEN (Production)
Bearer's Production API Key should be used only for production
X-API-TOKEN (Development)
Bearer's Development API Key should be used for testing
{{header}}
